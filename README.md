# SIGGAB

Sistema de Gestión (SIGGAB) desarrollado como proyecto académico.

## 📌 Descripción

Aplicación que integra un sistema backend y frontend para la gestión de información, con enfoque en escalabilidad y organización modular.

## 🧱 Estructura del proyecto

```
SIGGAB/
├── Backend/   # API, lógica de negocio
├── Frontend/  # Aplicación cliente
```

## 🚀 Tecnologías utilizadas

### Backend

* Node.js
* Express

### Frontend

* React Native / Web

## ⚙️ Instalación

### Backend

```
cd Backend
npm install
npm run dev
```

### Frontend

```
cd Frontend
npm install
npm start
```
